# Table

一个基于 Element-Plus 二次开发的表格组件，支持 Typescript、分页等模块。

可以传入json格式的数据 coloumns、data 来自定义表格展示的数据。

【文档详见Docs目录】

## 注意事项
由于storybook的限制，【Show code】中展示的代码，slots部分是经过编译后的代码，实际使用中用tsx的写法就行

比如Show code中展示如下代码：

```tsx
{
    "label": "签约类型",
    "prop": "type_cn",
    "slots": {
        "default": "(
        {
            row: e
        }
        )=>o(f(\"el-tag\"),
        {
            type: \"success\"
        },
        {
            default: ()=>
            [
                i(\" \"),
                e.order_id+1
            ]
        }
        )"
    }
}
```
实际上是：
```tsx
{
    label: '签约类型',
    prop: 'type_cn',
    slots: {
      default: ({ row }) => {
        return (<el-tag type="success"> {row.type_cn}</el-tag>)
      },
},
```
